# Todo App - Frontend

## Công Nghệ Sử Dụng

* **Core:** ReactJS (Create React App), React Router DOM (v6).
* **HTTP Client:** Axios.
* **Styling:** CSS Modules (Scoped CSS).
* **Notifications:** React Toastify.

## Yêu Cầu Tiên Quyết

* **Node.js**: Phiên bản 14.x trở lên.
* **npm**
* **Backend**: Đảm bảo Backend (folder `be`) đang chạy

## Hướng Dẫn Cài Đặt

cd fe
npm install
npm start
